# مشروع موقع مطعم شاورما أورا (Aura Shawarma)

موقع إلكتروني سينمائي وفاخر مصمم خصيصاً لمطاعم الشاورما والبيسترو الحديثة بأعلى معايير الـ UI/UX والأداء.

### 🌟 الهوية البصرية والتصميم (UI/UX)
- **الهوية البصرية**: تصميم عصري بالنمط الفاتح (Light Theme Only) يمزج بين الأبيض الناصع، درجات البرتقالي الجذاب (#F97316)، والأرقواني الدافئ والرمادي الداكن.
- **التجربة الرقمية**: تجربة سلسة متوافقة بالكامل مع الهواتف المحمولة والأجهزة اللوحية (Mobile First Responsive Design).
- **الحركات والتفاعلات**: انسيابية تامة في الانتقالات والتحويم باستخدام مكتبة Framer Motion مع شاشة تحميل أولية مبتكرة بدوران شعار المطعم.

---

### 🚀 الميزات والخصائص الرئيسية

1. **الهيدر الثابت والقائمة**:
   - شعار المطعم بخلفية مضيئة.
   - روابط تنقل سريعة وسلسة بين الأقسام.
   - زر مباشر لفتح نافذة رمز الـ QR.
   - زر اطلب عبر الواتساب.
   - قائمة جانبية متجاوبة للهواتف (Mobile Drawer Menu).

2. **القسم الرئيسي (Hero Section)**:
   - صورة عالية الجودة للأطباق الرئيسية.
   - مؤشرات الجودة والتقييمات (4.9/5 بناءً على 3500+ تقييم).
   - أزرار اتخاذ القرار (تصفح المنيو، اتصل بنا، الطلب عبر الواتساب).

3. **قصة المطعم (About Section)**:
   - استعراض مزايا الشواء على الحطب والخبز الطازج يومياً.
   - بطاقات توضيحية لسر النكهة والصلصات الحرفية.

4. **باقات العروض الخاصة (Offers Section)**:
   - عرض وجبات الغداء وصناديق الجمعات العائلية مع نسب التخفيض وأزرار الطلب المباشر.

5. **القائمة الرقمية التفاعلية (Digital Menu)**:
   - تصفية حسب التصنيف (لفائف الصاج، الأطباق، صناديق العائلة، المقابلات، العصائر، الحلويات).
   - محرك بحث فوري.
   - زر تصفية "الأكثر طلباً فقط" (Best Sellers).
   - نافذة تفاصيل الوجبة (Food Modal) تشمل السعرات الحرارية، نسبة الفلفل، المكونات، ورابط الطلب المباشر عبر الواتساب.

6. **معرض الصور التفاعلي (Interactive Gallery)**:
   - استعراض الأطباق والمطعم مع إمكانية التكبير والتصفح بسلاسة (Lightbox Viewer).

7. **تقييمات وآراء العملاء (Customer Reviews)**:
   - بطاقات التقييمات بشارات العميل الموثق والنجوم.

8. **رمز QR التفاعلي (QR Code Modal)**:
   - نافذة منبثقة برمز QR بدقة ناقشية لتصفح المنيو من الطاولة أو السيارة مباشرة.

9. **الفرع وخريطة قوقل (Google Maps & Contact)**:
   - إطار خريطة قوقل التفاعلية.
   - جدول ساعات العمل بالتفصيل.
   - نموذج تواصل تفاعلي يتيح للزوار إرسال الاستفسارات والرسائل مباشرة.

10. **صفحة الاتصال المخصصة (`/contact`)**:
    - صفحة مستقلة بكامل تفاصيل الفرع وقسم الأسئلة الشائعة (FAQ Accordion).
