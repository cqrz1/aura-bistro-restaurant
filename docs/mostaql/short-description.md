تطوير موقع إلكتروني فاخر وجاهز للإنتاج لمطعم شاورما بيسترو أصيل (Aura Shawarma) باستخدام Next.js 16، React، TypeScript، Tailwind CSS، وFramer Motion. يشتمل على قائمة طعام رقمية تفاعلية مع تصفية سريعة وبحث فوري، رمز QR للمنيو، حاسبة عروض، نظام آراء العملاء، خريطة قوقل التفاعلية، وتصميم متوافق 100% مع الهواتف الذكية.
